## 2. Intended Purpose

Proxy Pulse is designed and distributed exclusively for the following **lawful purposes**:

1. **Enterprise Network Infrastructure Management** — Managing, monitoring, and load-balancing corporate proxy server pools within authorized network environments.
2. **Network Quality-of-Service (QoS) Monitoring** — Testing proxy latency, availability, and throughput for compliance with internal SLAs or external service agreements.
3. **Software Testing and Quality Assurance** — Validating application behavior across different network paths and proxy configurations as part of development and QA workflows.
4. **Academic and Security Research** — Studying proxy protocols, network performance patterns, and connectivity behavior for legitimate academic or authorized security research.
5. **Content Delivery Verification** — Ensuring CDN nodes and edge servers are accessible and performing correctly across distributed infrastructure.
